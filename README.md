# Library Management System

Web-based Library Management System built with Google Apps Script.

## Features

- Dashboard
- Book Search
- Borrow Book
- Return Book
- Inventory Management

## Tech Stack

- Google Apps Script
- JavaScript
- HTML
- CSS
- Google Sheets
